# EventFlow Starter

免費的活動報名管理系統 — 自動彙整報名、管理名額、追蹤付款，把時間還給活動。

## 一句話介紹

把分散在 LINE、電話、紙本的報名資訊，全部集中到 Google Sheet，不用再熬夜對帳。

## 誰適合用

- 辦歌唱比賽、路跑、社區活動的主辦方
- 學校社團、系學會
- 里長辦公室、社區發展協會
- 任何需要收報名、對帳、管候補的人

## 功能

| 功能 | 說明 |
|------|------|
| 自動彙整報名 | 報名者填表後，資料自動進 Google Sheet |
| 名額上限管理 | 超過自動進候補，不會超收 |
| 報名確認 Email | 自動寄送確認信給報名者 |
| 付款狀態追蹤 | 手動標記已付款，誰還沒付一目瞭然 |
| 候補名單排序 | 自動按時間排序，不用自己記順序 |
| 匯出名單 CSV | 一鍵匯出，方便處理 |
