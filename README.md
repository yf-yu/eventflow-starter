# 🍜 食緣隨機

> 不知道吃什麼？讓命運決定吧~
> 我不難搞我是真的都可以啊!

免費 隨機 餐廳 推薦 工具，取的位子後一鍵決定好。

## 🎯 怎麼用？

1. 點**取得位置**，允許瀏覽器使用你的位置
2. 設定你想吃或不想吃的類型
3. 按**命運之輪**，交給命運決定

👉 [立即使用](https://hengzhenfang.pages.dev/tools/foodspin/)
